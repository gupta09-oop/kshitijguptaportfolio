## 📜 How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/gupta09-oop/kshitijguptaportfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd kshitijguptaportfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the project:
   ```bash
   npm start
   ```
